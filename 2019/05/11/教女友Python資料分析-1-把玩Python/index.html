程式說明

接著，我們一行行來看上面的程式怎麼執行。再看一次上面的程式碼：

  • 第1行，# hello.py是不必要的，這行只是告訴妳我把程式寫在hello.py這個檔案裡面。妳以為這是我自己慣用的語法嗎？當然不是！

    在Python中，#字號後面直到換行之前的東西都被稱為註解（comment），註解是用來向其他人說明你的程式如何運行的或是附加一些其他訊息（如：版權資訊）。簡而言之，他是給人看的資訊，不會被電腦處裡。若觀察仔細的話，第2行也有註解！(包含#之後直到程式結束)

  • 第2行，扣掉註解後是print('Hello, World!')，print()用來將括弧內的東西輸出到螢幕上，這裡是'Hello, World!'。

    也許妳會問，輸出的時候'Hello, Wolrd!'兩邊的單引號怎麼消失了呢？Python中，用引號包住的一系列文字稱為字串，用引號包著的目的是為了區分程式碼和「字串的內容」。
